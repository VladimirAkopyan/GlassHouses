Development Summary

Greenwich Millennium Village is a mixed-tenure modern development on the Greenwich Peninsula, planned as an urban village and part of the Millennium Communities Programme. The archive evidence points to an ambitious early-2000s sustainability brief, community heat network, landscaped courtyards, strong transport links, and a long record of later due-diligence issues around external-wall remediation, service charges, acoustic complaints, and infrastructure ageing.

How Answers Are Produced

The backend retrieves relevant chunks from MongoDB. If a Voyage API key is configured it uses Atlas Vector Search over the existing 1024-dimensional embeddings; otherwise it falls back to Mongo text search. Claude then writes the answer using only the retrieved passages and cites source ids.
